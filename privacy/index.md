 Privacyverklaring

Het Instituut voor Toegepaste Filosofie hecht veel waarde aan een zorgvuldige omgang met persoonsgegevens.  
Op deze website worden zo min mogelijk persoonsgegevens verwerkt.

 1. Wie zijn wij
Instituut voor Toegepaste Filosofie  
E-mail: info@instituutvoortoegepastefilosofie.nl

 2. Welke gegevens verwerken wij
Deze website verzamelt geen persoonsgegevens via formulieren.

Bij het bezoeken van de website kunnen wel technische gegevens worden verwerkt, zoals:
- IP-adres
- type browser
- bezochte paginaâ€™s

Deze gegevens worden verwerkt via de technische infrastructuur van de hostingpartij en zijn niet herleidbaar tot individuele personen door ons.

 3. Doel van de gegevensverwerking
De technische gegevens worden uitsluitend gebruikt voor:
- het technisch functioneren van de website
- beveiliging en stabiliteit van de website

 4. Delen van gegevens met derden
De website wordt gehost via GitHub Pages, een dienst van GitHub Inc. (Verenigde Staten).

GitHub kan technische gegevens verwerken in het kader van hosting en beveiliging.  
Het Instituut voor Toegepaste Filosofie maakt geen gebruik van tracking-, marketing- of advertentiediensten.

 5. Cookies
Deze website gebruikt geen cookies.

 6. Bewaartermijn
Technische gegevens worden bewaard zolang dit noodzakelijk is voor het functioneren en de beveiliging van de website, conform de bewaartermijnen van de hostingpartij.

 7. Jouw rechten
Je hebt het recht om:
- inzage te krijgen in jouw persoonsgegevens
- gegevens te laten corrigeren of verwijderen
- bezwaar te maken tegen verwerking

Voor vragen of verzoeken kun je contact opnemen via het genoemde e-mailadres.

 8. Wijzigingen
Deze privacyverklaring kan worden aangepast bij wijzigingen in de website of wetgeving.

Laatste wijziging: 5 januari 2026
